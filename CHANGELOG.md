# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog, with release entries grouped by version.

## [0.0.1] - 2026-03-23

### Added
- Added iOS 26-oriented toolbar controls with Expo UI SwiftUI on iOS.
- Added Android fallback controls, search field, and settings controls with improved contrast.
- Added city search backed by Open-Meteo geocoding.
- Added GitHub Actions workflows for Android build and OTA updates.
- Added EAS configuration for `preview` and `production` channels.

### Changed
- Refined weather home layout, settings layout, and city search interaction flow.
- Standardized app identifiers to `dev.wangxu.weatherrn`.
- Set the app version to `0.0.1` and aligned runtime versioning with app version.

### Fixed
- Stabilized rapid navigation between search and settings.
- Improved active saved-city deletion behavior in city search.
